Sales Performance Dashboard — AdventureWorks

This project analyzes AdventureWorks sales data and provides insights into revenue trends, regional performance, and product contribution using an interactive Power BI dashboard.

Project Objectives

    Track overall sales performance using KPI metrics

    Analyze yearly sales trends and demand patterns

    Identify top-performing regions and product categories

    Highlight top-revenue contributing products

Tech Stack

    Power BI

    SQL Server (AdventureWorks DW)

    Data Modeling (Star Schema)

    Dashboard Features

KPI Metrics

    Total Sales

    Total Orders

    Total Quantity Sold

Visual Insights

    Sales Trend by Year

    Sales by Region

    Sales by Product Category

    Top 10 Products by Sales

Interactive Filters

    Year

    Region

    Product Category

Key Business Insights

    Australia is the highest-revenue region

    Bikes category is the primary revenue driver

    Sales peaked in 2013 and dropped in 2014

    Majority of revenue comes from top bicycle models
